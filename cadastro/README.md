# Cadastro de usuário - React + Vite

## Sobre o Projeto

Aplicação de cadastro de usuários desenvolvida com React utilizando Vite.

O sistema permite inserir dados em um formulário e visualizar os usuários cadastrados na interface.

Este projeto foi desenvolvido com o objetivo de praticar conceitos de componentização,  organização de código e manipulção de estado do React.

## Tecnologias utilizadas

- React
- Vite
- JavaScript
- HTML5
- CSS3

## Como executar o projeto

Clone o repositório

git clone https://github.com/nixon-crypto/portfolio.git

Entre na pasta "Cadastro".

Instale as dependências

npm install

Execute o projeto

npm run dev
